## 🚀 快速部署指南

### 1️⃣ 安装后端依赖

打开命令行，执行：

```bash
cd D:\M-A_SYSTEM\V1.0\server
npm install
```

### 2️⃣ 配置数据库

编辑文件：`D:\M-A_SYSTEM\V1.0\server\config\index.js`

```javascript
module.exports = {
  database: {
    user: 'sa',              // 数据库用户名
    password: '密码',         // 数据库密码
    server: 'localhost',
    database: 'M-A_OP_ODB',
    options: {
      encrypt: false,
      trustServerCertificate: true
    }
  },
  jwtSecret: '修改为你的密钥',
  port: 3005
}
```

### 3️⃣ 启动服务器

```bash
cd D:\M-A_SYSTEM\V1.0\server
npm start
```

看到以下信息表示成功：
```
Server running on port 3005
API Documentation: http://localhost:3005/api/info
```

### 4️⃣ 配置 Web 服务器（Nginx）

安装 Nginx 后，配置：

```nginx
server {
    listen 80;
    
    # 前端
    root D:/M-A_SYSTEM/V1.0/frontend;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # 后端 API 代理
    location /api/ {
        proxy_pass http://localhost:3005;
    }
}
```

### 5️⃣ 访问系统

- **前端**: http://localhost
- **后端 API**: http://localhost:3005/api
- **API 文档**: http://localhost:3005/api/info

## 🐛 常见问题

### Q1: 端口 3005 被占用

**解决**:
```bash
# 查找占用进程
netstat -ano | findstr :3005

# 终止进程
taskkill /F /PID <进程 ID>
```

### Q2: 数据库连接失败

**检查**:
1. SQL Server 服务是否运行
2. 数据库名称是否正确
3. 用户名密码是否正确
4. 防火墙是否允许

### Q3: 前端页面空白

**解决**:
1. 按 F12 打开开发者工具
2. 查看 Console 错误
3. 检查 Network 请求
4. 确认 API 地址配置
