## 一、CRM模块功能说明

### 1.1 客户管理（核心功能）

#### 客户列表
- 分页展示客户列表
- 支持关键词搜索、客户类型筛选、企业类别筛选
- 实时统计数据：客户总数、活跃客户、潜在客户、正式客户、公海/私海客户
- 支持新建客户、导入、导出

#### 客户新增
- 表单字段完整，包括：
  - 基本信息：客户编码（自动生成）、客户名称、客户简称、客户类型、企业类别、客户来源、所属行业
  - 制造业专属：设备类型、年采购量、合作等级、区域、主营设备
  - 联系信息：联系人、职务、电话、邮箱、地址
  - 企业信息：年营业额、员工数量、主营产品、公司网站
  - 银行信息：开户银行、银行账号、税号
  - 分配信息：负责销售、分配日期
  - 其他：备注、标签
- 客户类型：设备厂商、集成商、终端客户、代理商、外协厂、潜在客户、正式客户

#### 客户编辑
- 打开编辑弹窗时从数据库实时拉取最新数据
- 支持修改所有字段
- 更新后自动刷新列表

#### 客户删除
- 删除前二次确认
- 自动处理关联数据（联系人、跟进记录、文档）
- 删除后自动刷新列表和统计数据

#### 客户详情
- 完整展示客户所有信息
- Tab页切换：基本信息、联系人、跟进记录
- 支持编辑、删除、添加跟进记录

### 1.2 联系人管理
- 联系人列表展示
- 添加/编辑/删除联系人
- 字段：姓名、职务、电话、手机、邮箱、生日、爱好、备注、是否主要联系人

### 1.3 跟进记录
- 时间线形式展示跟进历史
- 跟进类型：电话沟通、邮件往来、拜访客户、产品演示
- 字段：跟进类型、主题、内容、下次计划日期、下次计划内容
- 操作后自动刷新

### 1.4 客户公海
- 公海/私海分类管理
- 客户领取功能
- 客户释放到公海
- 客户分配（分配给其他销售）

### 1.5 竞争对手管理
- 竞争对手列表
- 竞争对手详情
- 竞争对手表单（新增/编辑）

## 二、技术架构

### 2.1 前端技术栈
- Vue 3 + Composition API
- Element Plus UI
- Pinia 状态管理
- Vue Router
- Axios HTTP客户端

### 2.2 后端技术栈
- Node.js + Express
- MS SQL Server 数据库
- 事件驱动架构（EventBus）
- 审计日志（AuditLog）

### 2.3 接口规范
- RESTful API设计
- 统一响应格式：`{success, data, message, timestamp, version}`
- 全局异常捕获
- 统一错误码

## 三、数据同步机制

### 3.1 前端刷新策略
| 操作 | 刷新机制 |
|------|----------|
| 新增客户 | 跳转列表页，mounted重新加载 |
| 编辑客户 | 跳转列表页，mounted重新加载 |
| 删除客户 | 调用loadCustomers() + loadStatistics() |
| 添加联系人 | 调用loadContacts() |
| 添加跟进 | 调用loadActivities() + loadCustomer() |
| 客户领取/释放/分配 | 调用loadCustomers() + loadStatistics() |

### 3.2 响应检查规范
```javascript
if (response && response.success) {
  // 处理成功
}
```

## 五、运行说明

### 5.1 环境要求
- Node.js 18+
- MS SQL Server 2016+
- npm 8+

### 5.2 启动方式
```bash
# 后端启动
cd server
npm install
node server.js

# 前端启动
cd frontend
npm install
npm run dev
```

### 5.3 访问地址
- 前端：http://localhost:3000
- 后端API：http://localhost:3005
